Grid Layout

Design your new grid layout by listing out json objects specify color, opacity, and location.
For example: "{color: 'Red', opacity: '20%', height: 10, width: 6, x: 3, y:7}"
This object would create a Red rectangle of 20% opacity that covers a 10x6 grid squares where the top left corner is the 4th square from the left 8 squares down.

Additional Notes
- The top left grid square is at x=0, y=0, rather than x=1, y=1.
- Setting a height of -1 will cause the shape to extend downwards infinitely.
- Setting a width of -1 will cause the shape to extend to the right infinitely.
- The different objects should be comma-separated.

Full Example
{"color": "Red", "opacity": "20%", "height": -1, "width": 6, "x": 3, "y":0},
{"color": "Green", "opacity": "30%", "height": 30, "width": 10, "x": 10, "y": 15},
{"color": "Blue", "opacity": "50%", "height": 20, "width": -1, "x": 0, "y": 10}
